            |                                                                     
            |                                                                     
            v

|________________________________________________________________________|
            |                                                                     
            |                                                                     
            v                                                                     
|________________________________________________________________________|
| Step 4: Adjust Offer Price                                              |

            |                                                                     
            |                                                                     
            v

            |                                                                     
            |                                                                     
            v                                                                     
|________________________________________________________________________|

| - Options:                                                             |

| - Notification: Buyer receives response                                 |
|________________________________________________________________________|
            |                                                                     
            |                                                                     
            v

|   - Options:                                                           |
|     - Reserve It                                                       |

|     - Improve Offer                                                    |
